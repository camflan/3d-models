# Office Billy Accessories

Various accessories for an IKEA Billy bookshelf office setup: LED strip mounts, cable management, shelf hardware, and clips.

## Models

### LedClip (OpenSCAD)

Simple clip for holding LED strips in place.

![LedClip preview](exports/led_clip_preview.png)

**Key parameters:**

| Parameter | Default | Description |
|-----------|---------|-------------|
| `led_width` | 10mm | Width of the LED strip |
| `led_thickness` | 2mm | Thickness of the LED strip |
| `clip_height` | 2mm | Height of the clip above the surface |
| `clip_grip_depth` | 1.5mm | How much the clip grips the strip |
| `wall_thickness` | 1.5mm | Wall thickness |

### LEDMounts (FreeCAD)

LED strip mounting brackets. Multiple body variants for different mounting scenarios.

**Exports:** BodyHole, BodyPocket001, BodyPocket002, BodySketch002, Body-version5, Body-version6

### LED PSU Standoff

Standoff for mounting an LED power supply.

### OfficeBillyTopRailSupport

Top rail support bracket for Billy bookshelf.

### OfficeBillyShallowLEDStripMounts

Shallow LED strip mounts designed for Billy shelf depth.

### RailAngleCableSupport (FreeCAD)

Angled cable support that attaches to a rail.

### Shelf Bushings

Replacement/custom shelf bushings (2 variants).
