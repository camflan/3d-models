# Skadis T-Clip Accessories

Parametric accessories for IKEA Skadis pegboard using the [T-clip system](https://www.printables.com/model/256896-skadis-t-clip-system).

## Models

### drill_driver_holder

Wall-mounted holster for a drill/driver. Slanted open-top box with a grip slot cutout in the base for easy removal. Clips are automatically placed on the Skadis 40mm/20mm alternating grid pattern.

**Key parameters:**

| Parameter | Default | Description |
|-----------|---------|-------------|
| `depth` | 160mm | How deep the holster is |
| `inner_height` | 75mm | Interior height |
| `inner_width` | 72mm | Interior width |
| `grip_slot_width` | 40mm | Width of the bottom grip cutout |
| `wall_thickness` | 4mm | Shell thickness |
| `slant_angle` | 65° | Front opening angle |
| `only_outer_clips` | false | Limit to corner clips only |

**Dependencies:** `clip-seat_fixed.stl` (included — sourced from the T-clip system, repaired via Formware)

## Printing

- No supports needed for the drill holder
- Recommend 3+ perimeters for clip strength
