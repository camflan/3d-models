# Fiona 8th Birthday Party

Two-color nameplate for Fiona's 8th birthday party.

## Models

### FIONA

![FIONA preview](exports/FIONA_preview.png)

Extruded text nameplate with a Minkowski-rounded border, designed for dual-color printing. Inner text extrudes taller than the border so a filament swap or multi-material setup produces a contrasting inlay.

**Key parameters:**

| Parameter | Default | Description |
|-----------|---------|-------------|
| `size_mm` | 130mm | Overall text size |
| `border_width` | 3mm | Border thickness around letters |
| `height` | 4mm | Base/border extrusion height |
| `letter_spacing` | 1.25 | Inter-letter spacing multiplier |

**Font:** American Typewriter

## Printing

- Print border layer first, pause/swap filament for inlay color
- No supports needed
